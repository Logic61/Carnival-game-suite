# 🎡 Carnival Game Suite (嘉年华游戏馆)

这是一个基于原生 HTML5 和 JavaScript 开发的轻量级博弈小游戏合集。无需后端，即开即玩。

---

## 🎮 包含的游戏

本项目集成了四款经典的随机性小游戏：

1.  **嘉年华骰子 (Carnival Dice)** - 模拟经典的嘉年华投掷体验。
2.  **老虎机 (Slot Machine)** - 经典的拉霸体验，看看你能不能转出三个相同图标。
3.  **比大小 (High or Low)** - 纯粹的运气对决，预测下一张牌或点数的大小。
4.  **猜硬币 (Coin Flip)** - 50/50 的直觉挑战。

---

## ✨ 项目特点

* **轻量化：** 纯原生前端技术堆栈（HTML/CSS/JS），无臃肿框架。
* **响应式：** 适配电脑与手机端，随时随地开启挑战。
* **零部署成本：** 完美适配 GitHub Pages。
* **随机公正：** 使用 JavaScript 内置随机引擎确保游戏随机性。

---

## 🚀 部署步骤

1. **Fork 本仓库**。
2. 进入仓库设置 **Settings** -> **Pages**。
3. 在 **Build and deployment** 部分，选择 `main` 分支并保存。
4. 等待几分钟，你的游戏馆就会在 `https://<your-username>.github.io/<repo-name>/` 上线。

---

## 🛠️ 技术实现

* **Markup:** HTML5
* **Styling:** CSS3 (Flexbox/Grid 布局)
* **Logic:** Vanilla JavaScript (原生脚本)
